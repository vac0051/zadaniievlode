# Japanese Multiplication Algorithm in Excel (VBA)

Этот проект содержит макрос на языке VBA, который реализует алгоритм японского (линейного) метода умножения двух двузначных чисел в виде наглядной таблицы прямо на листе Excel.

## 🛠 Как это работает

Алгоритм использует сетку ячеек (начиная с `F2`) для визуализации "пересечения прямых линий":
- **Красные линии (горизонтальные и вертикальные)** обозначаются цифрой `1`.
- **Синие линии (горизонтальные и вертикальные)** обозначаются цифрой `2`.
При наложении линий в таблице происходит их математическое сложение:
- `2` (1+1) — пересечение двух красных линий.
- `3` (1+2 или 2+1) — пересечение красной и синей линий.
- `4` (2+2) — пересечение двух синих линий.

Код подсчитывает количество двоек, троек и четверок, а затем по формуле `(количество двоек * 100) + (количество троек * 10) + (количество четверок)` выдаёт готовый результат умножения.

## 🚀 Инструкция по использованию

1. **Подготовьте лист Excel:**
   * В ячейку `A2` напишите **"Цифра 1"**.
   * В ячейку `B2` напишите **"Цифра 2"**.
   * В ячейку `C2` напишите **"Результат"**.
   * В ячейки `A3` и `B3` введите два двузначных числа, которые хотите перемножить (например, `19` и `49`).
   * Диапазон с `F2` по `W18` (и дальше) оставьте пустым — туда макрос будет выводить процесс работы алгоритма.

2. **Импорт кода:**
   * Нажмите `Alt + F11`, чтобы открыть редактор VBA.
   * Нажмите `File` -> `Import File...` (или `Файл` -> `Импорт файла...`).
   * Выберите файл `Multiplication.bas` из этой папки.

3. **Запуск:**
   * Вернитесь в Excel, нажмите `Alt + F8`.
   * Выберите макрос `MultiplyNumbers` и нажмите **"Выполнить"**.
   * Результат умножения появится в ячейке `C3`.
